## How install and use?

- For different CPU chip (Intel vs Apple Silicon), pick the matching download:
  - Apple Silicon (M1/M2/M3): files containing "arm64" (e.g., Global-Button-Sender-Mac-6.1.0-arm64.dmg or Global-Button-Sender-Mac-6.1.0-arm64-mac.zip)
  - Intel: files without "arm64" (e.g., Global-Button-Sender-Mac-6.1.0.dmg or Global-Button-Sender-Mac-6.1.0-mac.zip)
- Install and open (step-by-step):
  1) Download the correct file for your chip (see above)
  2) Open the downloaded .dmg or unzip the .zip and drag the app into /Applications
  3) Open Terminal and run (if macOS blocks it):
     sudo xattr -rd com.apple.quarantine "/Applications/Global Button Sender Mac.app"
  4) Launch the app from Applications
- To open the app if macOS shows a quarantine warning:
  sudo xattr -rd com.apple.quarantine "/Applications/Global Button Sender Mac.app"
